Both for High Definition and Lightweight rendering pipelines you need to have Unity 2019.1 or above. 
Universal Render Pipeline (URP) template works only with Unity 2019.3 or above.

Winter Environment was tested using HDRP versions from 5.7.2 to 11.0.0. Keep in mind, the HDRP Template is currently still in development. The package can be non-functional with other HDRP versions. Please be patient, we will add new versions of HDRP in the next updates. 

For more information about HDRP, LWRP and URP, please read the online documentation: https://docs.google.com/document/d/1JuJERUs9bHl0VlrwWS2Npzt2NnEEhCTISLQhiKrBZ54
If you have any questions and concerns do not hesitate to contact us at: contact@angrymesh.com

HDRP - HOW TO USE:
1. Create a new empty project using HDRP Template.
2. Go to Windows > Package Manager and select High Definition RP.
	- For Unity 2020.2.x or for a newer version of unity, update the High Definition RP to version 10.2.2 or above.
	- For Unity 2019.3.6.x, update the High Definition RP to version 7.2.1 or above.
	- For Unity 2019.3.x, update the High Definition RP to version 7.1.5 or above.
	- For Unity 2019.2.x, update the High Definition RP to version 6.9.0 or above.
	- For Unity 2019.1.x, update the High Definition RP to version 5.7.2 or 5.13.0.
3. Import Procedural Sky (Windows > Package Manager and select High Definition RP)
4. Import the Winter Environment package in the project.
5. Import the HDRP Template from ANGRY MESH > Nature Pack - Winter Environment > SRP. 

Note: 
For Unity 2020.2.x or newer, HDRP version 10.2.2 or above - You need to use HDRP template 10.2.2.unitypackage
For Unity 2019.3.6.x, HDRP version 7.2.1 or above - You need to use HDRP template 7.2.1.unitypackage
For Unity 2019.3.x, HDRP version 7.1.5 or above - You need to use HDRP template 7.1.5 (Legacy).unitypackage
For Unity 2019.2.x, HDRP version 6.9.0 or above - You need to use HDRP template 6.9.0 (Legacy).unitypackage
For Unity 2019.1.x, HDRP version 5.13.0 - You need to use HDRP template 5.13.0 (Legacy).unitypackage
For Unity 2019.1.x, HDRP version 5.7.2 - You need to use HDRP template 5.7.2 (Legacy).unitypackage

URP - HOW TO USE:
1. Create a new empty project using URP Template.
2. Go to Windows > Package Manager and select Universal RP.
	- For Unity 2020.2.x or for a newer version of unity, update the Universal RP to version 10.2.2 or above.
	- For Unity 2019.3.6, update the Universal RP to version 7.2.1 or above.
	- For Unity 2019.3.x, update the Universal RP to version 7.1.5 or above.
3. Import the Winter Environment package in the project. 
4. Import the URP Template from ANGRY MESH > Nature Pack - Winter Environment > SRP. 

Note: 
For Unity 2020.2.x or newer, URP version 10.2.2 or above - You need to use URP template 10.2.2.unitypackage
For Unity 2019.3.6.x, URP version 7.2.1 or above - You need to use URP template 7.2.1.unitypackage
For Unity 2019.3.x, URP version 7.1.5 or above - You need to use URP template 7.1.5 (Legacy).unitypackage

--------------------------------------------------------------------------------

LWRP (Legacy) - HOW TO USE:
1. Create a new empty project using LWRP Template.
2. Go to Windows > Package Manager and select Lightweight RP.
	- For Unity 2019.2.x, update the Lightweight RP to version 6.9.0 or above.
	- For Unity 2019.1.x, update the Lightweight RP to version 5.7.2.
3. Import the Winter Environment package in the project. 
4. Import the LWRP Template from ANGRY MESH > Nature Pack - Winter Environment > SRP > Legacy. 

Note: 
For Unity 2019.2.x, LWRP version 6.9.0 or above - You need to use LWRP template 6.9.0 (Legacy).unitypackage
For Unity 2019.1.x, LWRP version 5.7.2 - You need to use LWRP template 5.7.2 (Legacy).unitypackage
For Unity 2019.3.x, please use the URP Template.
